# Translation Prompt Template
# Version: 1.0
# Last Updated: 2025-11-02
#
# This template uses double-brace syntax (e.g., book_title in double braces)
# for simple string substitution.
#
# Available variables (in double braces):
#   book_title - Title of the book being translated
#   source_language - Source language (typically "English")
#   target_language - Target language (e.g., "Spanish", "Español")
#   source_text - The text chunk to translate
#   glossary - Formatted glossary terms for consistent translation
#   style_guide - Style guide content (tone, formality, dialect preferences)
#   context - Book context (genre, time period, style notes)
#   chapter_info - Optional chapter information for additional context
#   previous_chapter_context - Context from end of previous section (Original source language; do not re-translate)
#
# Instructions:
# - All variables must be provided when rendering this template
# - Use format_glossary_for_prompt() to prepare the glossary variable
# - StyleGuide.content should be used for the style_guide variable

================================================================================

================================================================================

You are translating the classic book "{{book_title}}" from {{source_language}}
to {{target_language}}. This is a professional translation for publication.

{{source_text}}

{{glossary}}

================================================================================
STYLE GUIDE
================================================================================

Follow these style guidelines for this translation:

{{style_guide}}

================================================================================
BOOK CONTEXT
================================================================================

{{context}}

{{previous_chapter_context}}

================================================================================
TRANSLATION INSTRUCTIONS
================================================================================

1. STRUCTURE PRESERVATION:
   - Preserve all paragraph breaks exactly as they appear in the source, do not merge or split paragraphs
   - The only exception would be to conform to standard dialogue handling in {{target_language}}, if necessary
   - If the source contains image placeholders in the format [IMAGE:filename], copy
     them into the translation exactly as-is at the same position in the text

2. GLOSSARY COMPLIANCE:
   - Use the exact translations provided in the glossary for all specified terms
   - Pay special attention to character names, place names, and key concepts
   - If a glossary term has alternatives, you may use them when appropriate
   - Do not translate glossary terms differently than specified

3. STYLE AND TONE:
   - Follow the style guide above exactly.

4. TRANSLATION QUALITY:
   - Translate for meaning, not word-for-word
   - Ensure natural-sounding {{target_language}}
   - Preserve literary devices (metaphors, allusions, wordplay when possible)
   - Maintain cultural context while making it accessible to target audience
   - Ensure grammatical correctness in {{target_language}}

5. PUNCTUATION AND FORMATTING:
   - Preserve intentional formatting (emphasis, capitalization for effect)
   - Adapt punctuation to {{target_language}} conventions
   - Keep any special formatting like letters, signs, or inscriptions

6. LENGTH AND COMPLETENESS:
   - Translate the complete text - do not skip or summarize
   - {{target_language}} translations are typically 10-30% longer than English
   - This is normal and expected - do not artificially compress the translation

================================================================================
OUTPUT FORMAT
================================================================================

Provide ONLY the translated text in {{target_language}}.

Do not include:
- Explanations or notes
- The original English text
- Commentary on translation choices
- Section headers or labels

Simply provide the clean translated text with proper paragraph breaks.
